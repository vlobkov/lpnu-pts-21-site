Типи скейтбордів

Класичний

Для трюків та міських вулиць
  • Вузька підвіска для чуттєвого контролю
  • Жорсткі колеса (99-101A) для ковзання по перилам
  • Ідеальний для: гріндов, фліпів, стійок на задніх колесах

Круїзер

Для комфортних міських подорожей
  • Широкий дек з загнутим хвостом
  • М'які колеса (78-85A) для гладкого ковзання
  • Ідеальний для: поїздок до кав'ярні, навчання основ

Лонгборд

Для рамп та скейт-парків
  • Міцна конструкція з широкою дошкою
  • Середні колеса (90-97A) для стабільності
  • Ідеальний для: в'їздів на вертикальні стінки, повітряних трюків
